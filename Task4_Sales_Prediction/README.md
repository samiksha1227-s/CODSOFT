#Sales Prediction Using Python

#Project Overview
  The project focuses on predicting product sales using machine learning techniques in Python. Sales prediction helps business make informed decisions by analyzing the impact of advertising expenditure on sales performance.

#Technologiess Used
  Python 
  Pandas
  Numpy
  Matplotlib
  Scikit-learn

#Module Used
  Linear Regression

#Output
  The module predicts sales values based on advertising expenditure and displays the evaluation metrics to measure accuracy.

#Conclusion
  This project demonstrates how machine learning can be applied to predict sales and assist business in optimizing their advertising strategies.
