# Credit Card Fraud Detection using Python

# Project Overview
  This project focus on detecting fraudulent credit card transactions using Machine Learning.
  The Goal is to classify transactions as "Fraudlent" based on transaction features.

# Technologies Used
  Python
  Pandas, NumPy
  Scikit-learn
  Logistic Regression

# Conclusion
  This project demonstrates how machine learning can be used to detect fraudlent transactions and highlights the importance of evaluation metrices when working with inbalanced datasets.
